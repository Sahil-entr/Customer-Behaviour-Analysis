# 📊 Customer Shopping Behavior Analysis

A complete end-to-end **Data Analytics Project** that analyzes customer shopping behavior using Python, SQL, and Power BI. This project demonstrates the complete analytics workflow—from data cleaning and exploratory analysis to SQL-based business insights and interactive dashboard creation.

---

## 📌 Overview

The objective of this project is to analyze customer purchasing behavior and generate actionable business insights. The project covers the entire data analytics pipeline, including:

- Data loading and preprocessing using Python
- Exploratory Data Analysis (EDA)
- Data cleaning and feature engineering
- SQL-based business analysis
- Interactive dashboard development in Power BI
- Business report documentation
- Project presentation created using Gamma

---

## 📂 Dataset

The dataset contains customer shopping transaction data, including:

- Customer Demographics
- Product Categories
- Purchase Amount
- Subscription Status
- Discounts & Promo Codes
- Shipping Type
- Customer Ratings
- Purchase Frequency

**Dataset Size**
- Rows: 3,900
- Columns: 18

## 🚀 How to Run

### Clone the Repository

```bash
git clone https://github.com/yourusername/customer-shopping-behavior-analysis.git
```

### Install Required Libraries

```bash
pip install pandas numpy matplotlib seaborn sqlalchemy
```

### Run the Project

1. Open the Jupyter Notebook.
2. Execute all cells in sequence.
3. Load the cleaned dataset into PostgreSQL, MySQL, or SQL Server.
4. Run the provided SQL queries.
5. Open the Power BI dashboard (`.pbix`) to explore interactive visualizations.
